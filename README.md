# Emotional Petri Dish

> A collection of lightweight HTML tools for emotional observation and self-reflection.
> 
> 一系列以情緒觀察與自我反思為核心的輕量化 HTML 工具。

Emotional Petri Dish is a collection of browser-based HTML applications created by **Ryesen** to support emotional awareness, self-observation, and personal growth.

### Included Tools
* 🌱 **Glimmer Greenhouse** — Cultivate and record emotional growth.
* 💻 **Terminal Project** — Organize thoughts and emotional reflections.
* 📊 **ERCA** — Review and analyze emotional events.

### Features
* No installation required
* Works entirely in your browser
* Offline available
* Local storage only
* Privacy-friendly

### Privacy
No personal data is collected or uploaded.
All records remain stored locally on your own device.

**情緒培養皿** 是由 **Ryesen** 開發的一系列 HTML 工具，旨在協助使用者進行情緒觀察、自我紀錄與反思。

### 收錄工具
* 🌱 **Glimmer Greenhouse（微光溫室）**：培養與記錄情緒變化。
* 💻 **Terminal Project（終端計畫）**：整理思緒、記錄每日觀察。
* 📊 **ERCA（爾卡情緒根因分析平台）**：協助回顧與分析情緒事件。

### 功能特色
* 無需安裝
* 開啟瀏覽器即可使用
* 支援離線使用
* 所有資料皆儲存在本機
* 不蒐集任何個人資料

### 隱私
所有資料皆儲存在使用者自己的裝置中，本工具不會上傳、收集或分享任何資料。
